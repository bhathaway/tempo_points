Help

What is this?

Tempo Points is a practice tool for musicians. When working up a passage to a target tempo, proceeding from slow tempo to full speed one bpm at a time is inefficient. This tool generates sensible intermediate tempo markings between a start and end BPM.

How does it work?

Rather than stepping by a fixed number of BPM (which feels like bigger jumps at lower tempos), Tempo Points uses a geometric progression — each step increases by approximately the same ratio. This matches how humans perceive tempo changes: going from 60 to 72 feels like the same size jump as going from 100 to 120 (both are +20%).

Using the tool

Enter your starting tempo, target tempo, and how many intermediate steps you want. Results update live as you type or use the +/− buttons. The percentage shown on each line is the ratio increase from the previous tempo.
